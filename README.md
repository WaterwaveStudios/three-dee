# three-dee

3D strategy/base-building game prototype built in Unity 6 (URP). Inspired by Last War.

## Setup

1. Install Unity 6 via Unity Hub
2. Open this directory as a Unity project
3. Install Blender for 3D asset creation

## Structure

- `Assets/` — Unity assets (scripts, models, materials, scenes)
- `Blender/` — Blender source files
