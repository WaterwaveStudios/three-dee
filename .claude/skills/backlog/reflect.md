# Backlog — Reflection Prompts

## Look For
- Was progress.md out of sync with actual backlog file state? Did items get created/completed but progress.md wasn't updated?
- Were items listed in the wrong section (e.g., completed item still in Pending)?
- Did `/backlog verify` need to reconcile progress.md?

## Update SKILL.md When
- A new reconciliation pattern is discovered (e.g., items deleted but still in progress.md)
- The progress.md format needs additional fields or sections
- Edge cases in numbering or ordering cause confusion

## Examples to Capture
- Specific drift scenarios that were caught and fixed
- Patterns that commonly cause progress.md to go stale
