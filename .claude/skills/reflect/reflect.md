# Reflect — Reflection Prompts

## Look For
- Were any skills renamed this session? Check all cross-references were updated (CLAUDE.md, other skills that invoke it)
- Was a reflect.md brainstormed and created for a skill this session? Verify the prompts are specific enough, not generic "did anything go wrong?" questions
- Were steps added, removed, or reordered? Verify step numbering is consecutive and the report template in Step 7 matches the actual steps
- Did the reflect skill grow too large this session? Check if any section should be extracted or simplified
- When updating project files, did you duplicate knowledge that already lives in CLAUDE.md? Keep a single source of truth

## Update SKILL.md When
- A new step was added but the report template wasn't updated to match
- Step numbering got out of sync after insertions
- A new file convention was introduced but not documented
- The brainstorm flow in Step 6d produced better interaction patterns worth codifying

## Examples to Capture
- Successful reflect.md brainstorm interactions (what made the prompts good)
- Cases where skill-specific reflection found something the generic process missed
